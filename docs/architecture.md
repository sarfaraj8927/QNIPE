# Architecture Notes

This repository scaffold mirrors the folder structure and phase roadmap defined in
`QNIP_system_architecture.pdf` dated July 18, 2026.

## Phase 1 Deliverables

The architecture defines Phase 1 as:

1. initialize the GitHub repository
2. add `pyproject.toml`
3. add CI/CD and pre-commit hooks
4. create the folder structure
5. implement `PAQNIPEConfig`
6. implement `ExperimentLogger`

## Included in This Scaffold

- root packaging and contributor files
- configuration profiles under `configs/`
- data, docs, notebooks, scripts, and source placeholders
- unit, integration, and theorem-validation test placeholders

## Deferred to Later Phases

All algorithmic work remains deferred to later roadmap phases. Placeholder modules are present only
to preserve the architecture exactly.
