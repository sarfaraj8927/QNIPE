- Reproduce the published algorithm faithfully.
- Never redesign the algorithm.
- Never invent filenames or modules.
- Follow the architecture exactly.
- Implement one package at a time.
- Never modify unrelated files.
- Use Python 3.12+.
- Follow PEP8.
- Use type hints.
- Use dataclasses where appropriate.
- Use logging.
- Every module must include unit tests.
- Every public function must include docstrings.
- If the paper and architecture conflict, stop and explain before proceeding.
