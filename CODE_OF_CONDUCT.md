# Code of Conduct

## Our Standard

We expect contributors to:

- communicate respectfully and directly
- assume good intent while giving technical feedback
- keep disagreements focused on the work, not the person
- avoid harassment, discrimination, and personal attacks

## Reporting

If conduct within the project becomes a problem, report it to the project maintainers through the
repository's private maintenance channel.

## Scope

This code of conduct applies to repository discussions, pull requests, issue threads, and other
project collaboration spaces.
